Numerical Aspects for Approximating Governing Equations Using Data

Posted on 2022-12-18 In paper

Basic Information

  • Title: Numerical Aspects for Approximating Governing Equations Using Data
  • Authors: Kailiang Wu, Dongbin Xiu
  • Pub Date: 2019-05-01
  • Journal/Conference: Journal of Computational Physics

Overview

  • This work employs a set of standard basis functions, e.g. polynomials, to approximate the governing equation with high accuracy.
  • This work discusses the importance of using a large number of short bursts of trajectory data, rather than using data from a single long trajectories.
  • Several options for the numerical algorithms to perform accurate approximation are then presented, along with an error estimate of the final equation approximation

Background & Motivation

  • For all physical laws and models, they are approximation to the true physics.
  • Even though some numerically constructed laws or governing functions don't possess analytical form, if they are reasonably accurate, they can serve as good models.

Main Contributions of This Work

Main Focus: local recovery & approximation of general 1st-order nonlinear ordinary differential / algebraic equations from the time-varying measurement data.

This work: - seeks accurate approximation to the underlying governing equations, rather than exact recovery of governing equations. - Advantage: able to utilize many standard approximation methods using standard basis functions, e.g. polynomials. - proposes the use of multiple trajectory data of very short span, instead of using a single long trajectory data. - It needs to merely long enough to enable accurate estimate of the time derivatives - convert the equation recovery problem into a function approximation problem with over-sampled data. - discusses the results obtained by least squared method and \(\ell_{1}\) regression method.

Main Difference between This Work & Prior Ones

  • Early work: Symbolic Regression
  • Sparse-promoting techniques & Pareto analysis
    • Basic Idea: construct parsimonious governing data from a combinatorially large of candidate models
    • Applications:
      • recover PDE
      • infer structures of Network models
    • Variations:
      • Sparse Convex Optimization & In the space of multivariate polynomials
      • Nonconvex Sparse Optimization & Selection + Identification model from noisy data
      • Delay Embedding & Koopman Theory: decompose chaotic system \(\to\) forced linear system
  • Other techniques: sampling theory; motion from a data series; ANN; nonlinear Laplacian spectral analysis; modeling emergent behavior; automated inference of dynamics

Main Techniques

Problem Setup

For autonomous system in the form of \[ \frac{\mathrm{d} \boldsymbol{u}(t)}{\mathrm{d}t} = \boldsymbol{f}(\boldsymbol{u}), \quad \boldsymbol{u}(t_{0}) = \boldsymbol{u}_{0}, \] where - \(\boldsymbol{u} = (u_{1}, u_{2}, \dots, u_{d})^{\top}\) are state variables - \(\boldsymbol{f}\) defines the evolution of the states and is the true dynamics - \(\boldsymbol{u}(t;\boldsymbol{u}_{0})\) is a trajectory Our goal is to construct an accurate representation of \(\boldsymbol{f}\) using measurement data of the state variables \(\boldsymbol{u}\)

Domain of Interest

We restrict our equation recovery to a bounded domain \(D \subset \mathbb{R}^{d}\) and \(D\) is called domain-of-interest, which stands for the range of state variables where one focuses on the underlying problem.

Let \[ U_{i} = [u_{\min, i},u_{\max, i}],\quad 1 \le i \le d, \] be a closed interval and represent a range of the state variable \(u_{i}, ~1 \le i \le d\). We then define \(D\) as the product of these intervals \[ D = \times_{i=1}^{d} U_i. \]

  • We then equip it with a measure \(\omega\) which satisfies \(\int_{D} \mathrm{d}\omega = 1\).
  • We the invoke the standard definition of inner product \[ (g, h)_{L^{2}_{\omega}} := \int_{D} g(\boldsymbol{x}) h(\boldsymbol{x}) \mathrm{d} \omega(\boldsymbol{x}), \] and its induced norm \(\| \cdot \|_{L_{\omega}^{2}}\).

The domain-of-interest allows one to focus on a region of the domain where the problem can be well behaved and \(\boldsymbol{f}\) possesses a regular/smooth form, which would be highly advantageous for the recovery effort

Data

We assume that the observation data denote \[ \boldsymbol{x}(t_{j}; \boldsymbol{u}_{0}) = \boldsymbol{u}(t_{j}; \boldsymbol{u}_{0}) + \boldsymbol{\epsilon}_{j}, \quad j = 0, 1, \dots \] and \(\boldsymbol{\epsilon}_{j}\) stands for observation errors.

The trajectory data can be collected form different sequences of trajectories, which are originated by different initial states. \[ \boldsymbol{x}^{(m)}(t_{j}) = \boldsymbol{u}(t_{j}; \boldsymbol{u}_{0}^{m}) + \boldsymbol{\epsilon}_{j}^{(m)} \in D, \quad 0 \le j \le J_{m},~1 \le m \le M, \] where \(J_{m}\) is the number of intervals in the \(m\)-th trajectory data.

Extension to Differential-Algebraic Equations

The method discussed in this paper is applicable to semi-explicit differential algebraic equations (DAEs) in the following form: \[ \begin{cases} \frac{\mathrm{d} \boldsymbol{u}}{\mathrm{d}t} = \boldsymbol{F}(\boldsymbol{u}, \boldsymbol{v}),\\ \boldsymbol{G}(\boldsymbol{u}, \boldsymbol{v}) = 0, \end{cases} \] where - \(\boldsymbol{u}\), \(\boldsymbol{v}\): State Variables - \(\boldsymbol{G}\): Algebraic Constraints Assume that \(\boldsymbol{G}(\boldsymbol{u}, \boldsymbol{v}) = 0\) induces the following explicit relation \[ \boldsymbol{v} = \boldsymbol{g}(\boldsymbol{u}) \] Then by letting \(\boldsymbol{f}(\boldsymbol{u}) = \boldsymbol{F}(\boldsymbol{u}, \boldsymbol{g}(\boldsymbol{u}))\), the DAEs can be rewritten in the form of autonomous system.

Main Numerical Algorithm

General Formulation

We assume that the time derivatives of the trajectory are available at some time instances \(t_{j},~j \in \Theta_{m} \subset \{0, 1,\dots,J_{m}\}\). Let \[ \dot{\boldsymbol{x}}(t) = \dot{\boldsymbol{u}}(t) + \boldsymbol{\tau}(t), \] where \(\boldsymbol{\tau}(t)\) stands for the errors or noises. This creates data parings at the time instances \(\{t_{j}, j\in \Theta_m\}\), where both state variables and their derivatives are available. The collection of such pairings are denoted as \[ \left\{ \boldsymbol{x}^{(m)}(t_{j}), \dot{\boldsymbol{x}}^{(m)}(t_{j}) \right\}, \quad j \in \Theta_{m}, \quad 1 \le m \le M, \] where the total number of data paring is \[ M_{tot} = \sum\limits_{m=1}^{M}\#\Theta_{m}. \] We then choose a large set of dictionary \(\boldsymbol{\phi} = \{\phi_{1}, \dots, \phi_{N}\}\) to represent \(\boldsymbol{f}\) in the governing equation. For each \(\ell=1, \dots, d\), we seek \[ f_{\ell}(\boldsymbol{x}) \approx \tilde{f}_{\ell} (\boldsymbol{x}) = \sum\limits_{j=1}^{N} c_{j, \ell} \phi_{j}(\boldsymbol{x}) \] Let \(\boldsymbol{A}\) be a \((M_{tot}, N)\) model matrix

\[ \boldsymbol{A} = (a_{ij}), \quad a_{ij} = \phi_{j}(\boldsymbol{x}_{i}), \quad 1 \le i \le M_{tot}, \quad 1 \le j \le N, \]

where \((\boldsymbol{x}_{i}, \dot{\boldsymbol{x}}_{i})\) represents the \(i\)-th data paring. The equation recovery problem is then cast into an approximation problem: \[ \boldsymbol{A}\boldsymbol{c}_{\ell} = \dot{\boldsymbol{X}}_{\ell}, \quad \ell = 1, \dots, d, \] where \(\dot{\boldsymbol{X}}_{\ell}\) is the vector of \(\ell\)-th component of all available derivatives in the paring, and \(\boldsymbol{c}_{\ell} = (c_{1,\ell}, \dots, c_{N,\ell})^{\top}\) is the collected vector to be solved.

Once an accurate approximation is obtained, we consider \[ \frac{\mathrm{d}\boldsymbol{x}(t)}{\mathrm{d} t}= \tilde{\boldsymbol{f}}(\boldsymbol{x}), \] as an approximation of the true system.

Equation Approximation Using Standard Basis

  • The linear subspace of polynomials of degree up to \(n \ge 1\): \[ \Pi_{n}^{d} = \mathrm{span} \{ \boldsymbol{x}^{\boldsymbol{i}} = x_{1}^{i_{1}} \cdots x_{d}^{i_{d}}, |\boldsymbol{i}| \le n \} \] where \(\boldsymbol{i} = (i_{1}, \dots, i_{d})\) is multi-index with \(|\boldsymbol{i}| = i_{1} + \dots + i_{d}\) and \[ N = \dim \Pi_{n}^{d} = \begin{pmatrix} n+d \\ d \end{pmatrix} = \frac{(n+d)!}{n!d!}. \]

  • We seek \[ \tilde{f}_{\ell} (\boldsymbol{x}) = \sum\limits_{j=1}^{N} c_{j, \ell} \phi_{j}(\boldsymbol{x}) = \left\langle \boldsymbol{c}_{\ell}, \boldsymbol{\Phi}(\boldsymbol{x})\right\rangle, \quad \ell = 1, \dots, d \] where \[ \boldsymbol{\Phi(\boldsymbol{x})} = (\phi_{1}(\boldsymbol{x}), \dots, \phi_N(\boldsymbol{x}))^{\top}, ~\boldsymbol{c}_{\ell} = (c_{1,\ell}, \dots, c_{N,\ell})^{\top}. \] #### Selection of Trajectories

  • Key Facts: Data should fill up the underlying domain in a symmetric manner.

  • The trajectories are started from different initial conditions.

  • The length of each trajectories may be very short, just long enough to allow one to estimate the time derivatives.

Several Approaches: 1. Sampling the initial states \(\{ \boldsymbol{u}_{0}^{(m)} \}\) from random distribution defined on \(D\) 2. Sampling the initial states \(\{ \boldsymbol{u}_{0}^{(m)} \}\) by quasi-Monte Carlo sequences, such as the Sobol or Halton sequence, or other lattice rules 3. If \(D\) is hypercube, the initial state set \(\{ \boldsymbol{u}_{0}^{(m)} \}\) can be taken as the set of quadrature points (e.g. the Gaussian points) or its random subset. One can also sample the quadrature points according some discrete measure

Numerical Strategies

Computation of Time Derivatives

In most cases only the trajectory data \(\{\boldsymbol{x}^{(m)}(t_{j})\}\) are available.

For noiseless data: high-order finite difference method \[ \dot{\boldsymbol{x}}^{(m)}(t_{1}) = \frac{\boldsymbol{x}(t_{2}) - \boldsymbol{x}(t_{1})}{2\Delta t} \] For noise data: - Least Squares Approximation Methods (This Paper) - Tikhonov Regularization - Total Variation Regularization - Knowles and Wallace Variation Methods

Recovery Algorithms: Regression Methods

General Form:

\[ \boldsymbol{c}_{\ell} = \mathop{\arg\min}_{\boldsymbol{c} \in \mathbb{R}^{N}} \frac{1}{2} \| \boldsymbol{Ac} - \dot{\boldsymbol{X}}_{\ell} \|, \quad \ell = 1, \dots, d \] - \(\|\cdot\|\) is \(\ell_{2}\)-norm: Least Squares method - \(\|\cdot\|\) is \(\ell_{1}\)-norm: Least Absolute Deviation (LAD)

LASSO Regression: \[ \boldsymbol{c}_{\ell} = \mathop{\arg\min}_{\boldsymbol{c} \in \mathbb{R}^{N}} \frac{1}{2} \| \boldsymbol{Ac} - \dot{\boldsymbol{X}}_{\ell} \|^{2} + \lambda \|\boldsymbol{c}\|_{1} \]

Since this method here seeks accurate approximation to the governing equations, rather than exact recovery, we focus on the use of least squares and LAD.

Recovery Algorithms: Matrix-Free Method

\[ \boldsymbol{c}^{(k)}_{\ell} = \boldsymbol{c}^{(k-1)}_{\ell} + \frac {\dot{\boldsymbol{x}}_{\ell}(t_{jk}) - \langle \boldsymbol{c}^{(k-1)}_{\ell}, \boldsymbol{\Phi}(\boldsymbol{x}^{(k)}(t_{jk})) \rangle} {\| \boldsymbol{\Phi}(\boldsymbol{x}^{(k)}(t_{jk})) \|_{2}^{2} + \gamma_{k}} \boldsymbol{\Phi}(\boldsymbol{x}^{(k)}(t_{jk})), \quad k = 1, \dots, \]

Advantages & Shortcomings

  • Advantage: The concept of approximating equation rather than recovering the equation is great. This makes it possible to use approximation theory in this field.

  • Disadvantage: In applications we may actually need exact equation recovering.
